<p align="center">
  ✦ &nbsp;<strong>VesperaOS</strong>&nbsp; ✦
</p>

<p align="center">
  a freestanding x86_64 hobby kernel written in C++20 — built entirely from scratch. 🌊<br/>
  no stdlib. no shortcuts. custom memory manager, intel blitter driver, vfs, ahci driver, and more.<br/>
  <sub>just page tables, late nights, and a lot of QEMU restarts.</sub>
</p>
